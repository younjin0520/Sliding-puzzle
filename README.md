# Sliding-puzzle
### 1. 소개
  - 슬라이딩 퍼즐게임을 즐길 수 있는 안드로이드 애플리케이션입니다.
  - Java 언어로 안드로이드 스튜디오를 통해 작성되었습니다.

### 2. UI 및 게임 방법 설명
(1)메뉴 화면에서 ‘GAME START’ 버튼을 누릅니다
